# 腾讯云批量操作工具

这是一个使用Streamlit开发的腾讯云批量操作工具，可以实现以下功能：

- 批量开关机实例
- 批量创建镜像
- 批量删除镜像
- 批量创建快照
- 批量删除快照

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行程序：
```bash
streamlit run app.py
```

2. 在浏览器中打开显示的地址
3. 按照界面提示进行操作

## 注意事项

- 使用前请确保已正确配置腾讯云的认证信息
- 删除操作需要输入正确的密码
- 请谨慎操作，避免误删除重要资源
